## Project Screenshot

## Project Overview
This project analyzes Superstore sales data using PostgreSQL. The goal is to identify sales trends, profit performance, customer behavior, and category-wise performance.

## Dataset
- Superstore Sales Dataset
- Format: CSV

## Tools Used
- PostgreSQL
- SQL
- CSV Dataset

## Analysis Performed
- Total Sales Analysis
- Total Profit Analysis
- Category-wise Sales
- Region-wise Sales
- Top Customers
- Top Products
- Profit by Category

## Key SQL Concepts Used
- SELECT
- WHERE
- GROUP BY
- ORDER BY
- Aggregate Functions
- CASE Statements

## Files Included
- superstore_sales_analysis.sql
- superstore.csv
